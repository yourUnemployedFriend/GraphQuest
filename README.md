## Overview

Basic game where you have to guess the graphic function shown in the canvas. Right now its just completing "f(x) = ??".

## Requirements

A browser, that's it. It's just javascript.

## How to play

1. Look at the graph
2. Write the function that you thing f(x) equals to
3. Press "Draw"

## Legend
